## How to run locally ?
Step 1: Install `mysql`

Step 2: Run the sql create and alter command from `src/sql/create.sql`

Step 3: Run `npm install` to install dependencies

Step 4: Run `npm start` and navigate to `http://localhost:3000/stardust`
